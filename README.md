# لیست سرور های تیم اسپیک ایران

لیست برترین و معتبرترین سرورهای TeamSpeak ایران.

آدرس: https://list.tsserver.ir

## ویژگی‌ها

- نمایش لیست سرورهای تیم اسپیک ایران
- جستجو و مرتب‌سازی سرورها
- تم تاریک و روشن
- قابلیت اتصال مستقیم با `ts3server://`
- نمایش وضعیت آنلاین/آفلاین
- واکنش‌گرا (Responsive)
- سئوی بهینه
- دسترسی‌پذیری (Accessibility)
- PWA و نصب روی موبایل

## مشارکت

برای اضافه کردن سرور جدید، [اینجا](https://github.com/30j/list.tsserver.ir/issues/new?template=add-server.yml) یک Issue باز کنید.
